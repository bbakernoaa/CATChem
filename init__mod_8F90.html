Detailed Description

Contains the DiagStateType and Diag_Allocate subroutine.

Initialization module for the program.

This module contains subroutines and functions related to the DiagState instance of CATChem. It includes subroutines for initializing of the DiagState.

This module contains subroutines and functions related to the initialization of the program. It includes subroutines for initializing the grid, the time step, and the solution.
